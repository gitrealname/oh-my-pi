# Test Prompts

These templates are used for manual and automated testing of the m-prompt-template extension.
Deploy them before running tests:

```bash
# Deploy to global prompts dir:
cp test-prompts/*.md ~/.pi/agent/prompts/

# Deploy to project prompts dir (from D:/.ai):
cp test-prompts/*.md .pi/prompts/
```

Required skill for T5: `~/.pi/skills/test-marker.md` (see below).
